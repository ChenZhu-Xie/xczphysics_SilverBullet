# 功能
- 查部首，如 辶廴𧾷
>   与官鹤一样
  - 则：ob + z

# 使用心得

相比于 [[飞鹤快拼]]，墨奇 ==不需要 指定==：形码后缀 的 提示符
他在自动计算不同的排列组合下的最合乎表达逻辑的词组。

但有个问题：四码分字 的 命中率很低。
比如：==好嘟== 这个用 [[Language/Input Method/官鹤]] 很容易打出来的 双单字 “好 + 嘟”
- 在 [[Language/Input Method/墨奇]] 中就很难通过 hcduke 打出来。
  - 那么怎么造词
